# Retail-Data-Quality-Monitoring-System

## Overview

The Retail Data Quality Monitoring System is a data analytics project designed to assess, monitor, and improve the quality of retail product data. The project uses SQL, Python, PostgreSQL, and Power BI to identify data quality issues, calculate key performance indicators (KPIs), generate alerts, and visualize data quality metrics through interactive dashboards.

This project demonstrates practical skills in data validation, data quality management, business intelligence reporting, and automation.

## Objectives
* Monitor product data quality in a retail environment.
* Detect missing, duplicate, and invalid records.
* Calculate a measurable Data Quality Score.
* Automate quality monitoring and alert generation.
* Build dashboards for business users and stakeholders.

## Dataset
The project uses a retail or e-commerce product dataset containing product information.
